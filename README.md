# SoulSeesBest
IDM Experimental Electronica Rooted in the Southwestern US

This repo serves the AI Architecture Lab by WriterJoshua SoulSeesBest web app project. 

## Music
SoulSeesBest is an electronic instrumental project based in Southern California, Arizona, and New Mexico. Primary tools include GarageBand for iOS and MacOS, Casio CTK-2000, and various live instrumentation and custom sound effects.

## App
Working with AI tools, Claude Haiku 4.5 and VS Code, AI Architecture Lab is currently designing the official web application/website for the brand.
